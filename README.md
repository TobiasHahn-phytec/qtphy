qtphy – PHYTEC's Qt6 Reference Implementation
================================================================================

Building
--------------------------------------------------------------------------------

Build *qtphy* with the [Meson Build system](https://mesonbuild.com/):
```
meson setup build
meson compile -C build
```
After building, the resulting binary is located at `build/qtphy` and can be
directly executed.

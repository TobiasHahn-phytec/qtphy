qtphy – PHYTEC's Qt5 Reference Implementation
================================================================================

Building
--------------------------------------------------------------------------------

Build *qtphy* with `qmake` and `make`:
```
mkdir build
cd build
qmake ..
bindir=/usr/bin make
```
Make sure to specify `bindir` when executing `make`.
